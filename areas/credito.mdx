O **Motor de Crédito** integra a API Mycroft para análise automatizada de borderôs CNAB 400, com motores de regras configuráveis e transparência total sobre cada regra aplicada.

## O que você consegue fazer

- **Criar operações de análise** — Upload de arquivos CNAB 400 com XMLs opcionais de NFe
- **Acompanhar status** — Monitorar filas de processamento em tempo real
- **Consultar resultados** — Árvore hierárquica de cedente, sacados e títulos com issues detalhadas
- **Gerenciar engines** — Criar, clonar e personalizar motores de regras (blocking/alert)

  Acesse a tab **Crédito** para ver todos os endpoints disponíveis.
