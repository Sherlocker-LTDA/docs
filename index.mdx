O Mycroft e uma plataforma de analise de credito transparente para FIDCs (Fundos de Investimento em Direitos Creditorios) brasileiros. A API recebe arquivos CNAB 400, aplica regras de credito configuraveis, e retorna resultados hierarquicos com total transparencia sobre cada decisao.
