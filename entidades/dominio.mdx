Um **Domínio** representa um domínio de internet (website) encontrado via busca reversa WHOIS a partir dos emails associados a um CPF ou CNPJ.

## Tipagem

```json
{
  "documento": "12345678901",
  "total": 2,
  "dominios": [
    {
      "dominio": "exemplo.com.br",
      "registrado": true,
      "data_criacao": "2020-03-15",
      "data_atualizacao": "2024-01-10",
      "data_expiracao": "2025-03-15",
      "registrador": {
        "nome": "Registro.br",
        "url": "https://registro.br"
      },
      "registrante": {
        "nome": "Joao Silva",
        "empresa": "Empresa Ltda",
        "email": "joao@exemplo.com.br",
        "telefone": "+55.1199999999",
        "pais": "Brazil"
      },
      "email_origem": "joao@exemplo.com.br"
    }
  ]
}
```

## Campos

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `documento` | string | CPF ou CNPJ consultado |
| `total` | number | Total de domínios encontrados |
| `dominios` | array | Lista de domínios |

### Domínio

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `dominio` | string | Nome do domínio (ex: `exemplo.com.br`) |
| `registrado` | boolean | Se o domínio está registrado atualmente |
| `data_criacao` | string \| null | Data de criação do registro (ISO 8601) |
| `data_atualizacao` | string \| null | Data da última atualização do registro |
| `data_expiracao` | string \| null | Data de expiração do registro |
| `registrador` | object \| null | Dados do registrador (empresa que registrou) |
| `registrador.nome` | string \| null | Nome do registrador |
| `registrador.url` | string \| null | URL do registrador |
| `registrante` | object \| null | Dados do dono do domínio |
| `registrante.nome` | string \| null | Nome do registrante |
| `registrante.empresa` | string \| null | Empresa do registrante |
| `registrante.email` | string \| null | Email do registrante |
| `registrante.telefone` | string \| null | Telefone do registrante |
| `registrante.pais` | string \| null | País do registrante |
| `email_origem` | string | Email que originou a busca reversa |

## Como funciona

1. Busca todos os **emails** associados ao CPF/CNPJ no banco de dados
2. Para cada email, faz uma busca reversa WHOIS via [Whoxy](https://www.whoxy.com/) para encontrar dominios registrados com aquele email
3. Para cada dominio encontrado, busca os dados completos de WHOIS (registrador, registrante, datas)
4. Deduplica dominios que aparecem em mais de um email

## Endpoints

| Rota | Descrição |
|------|-----------|
| `GET /dominios/cpf/{cpf}` | Domínios vinculados a emails de uma pessoa |
| `GET /dominios/cnpj/{cnpj}` | Domínios vinculados a emails de uma empresa |
