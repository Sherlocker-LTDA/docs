Uma **Parte** representa uma pessoa física ou jurídica envolvida em um processo judicial.

## Campos

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `nome` | string | Nome da parte (Title Case) |
| `documento` | string | CPF ou CNPJ formatado (pode ser vazio) |
| `polo` | string | `ativo`, `passivo`, `terceiro`, `interessado` |
| `tipo` | string | Tipo de pessoa (ex: `Física`, `Jurídica`) |

## Exemplo

```json
{
  "nome": "João Silva de Oliveira",
  "documento": "123.456.789-01",
  "polo": "ativo",
  "tipo": "Física"
}
```

## Polos

| Polo | Descrição |
|------|-----------|
| `ativo` | Quem inicia a ação (autor, exequente, reclamante) |
| `passivo` | Contra quem a ação é movida (réu, executado, reclamado) |
| `terceiro` | Parte interessada que ingressa no processo |
| `interessado` | Interessado no processo |
