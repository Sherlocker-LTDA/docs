Uma **Licitacao** representa um processo de contratacao publica em que o CPF ou CNPJ participou como fornecedor, conforme registrado no PNCP.

## Campos

| Campo | Tipo | Descricao |
|-------|------|-----------|
| `id` | string (uuid) | ID interno da licitacao |
| `id_licitacao` | string | Numero da licitacao no sistema de origem |
| `id_pregao` | string | Numero do pregao (se houver) |
| `id_uasg` | string | Codigo UASG do orgao contratante |
| `titulo` | string | Titulo do edital |
| `descricao` | string | Objeto da licitacao |
| `fonte` | string | Fonte dos dados (`PNCP`) |
| `modalidade` | string | Modalidade de contratacao (ex: `Pregao Eletronico`, `Credenciamento`) |
| `tipo_disputa` | string | Tipo de disputa (ex: `Aberto`, `Fechado`) |
| `data_publicacao` | string (date) | Data de publicacao |
| `data_inicio` | string (date) | Inicio da vigencia |
| `data_fim` | string (date) | Fim da vigencia |
| `uf` | string | UF do orgao contratante |
| `local` | string | Municipio do orgao contratante |
| `exclusivoMEEPP` | boolean | Exclusivo para Microempresas e EPP |
| `items` | array | Itens do edital |
| `participantes` | array | Todos os participantes da licitacao |

## Items

| Campo | Tipo | Descricao |
|-------|------|-----------|
| `id` | string (uuid) | ID do item |
| `codigo` | string | Codigo do item no edital |
| `descricao` | string | Descricao do item |
| `quantidade` | number | Quantidade |
| `valor_total` | number | Valor total estimado em reais |
| `decreto7174` | boolean | Aplica preferencia Decreto 7174 |
| `exclusivoMEEPP` | boolean | Item exclusivo para ME/EPP |

## Participantes

| Campo | Tipo | Descricao |
|-------|------|-----------|
| `id` | string (uuid) | ID do participante |
| `documento` | string | CPF ou CNPJ |
| `nome` | string | Nome ou razao social |
| `tipo` | string | `PF` (pessoa fisica) ou `PJ` (pessoa juridica) |
| `totalLicitacoes` | number | Total de licitacoes do participante |
