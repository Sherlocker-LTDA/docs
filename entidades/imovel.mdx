Um **Imóvel** no Sherlocker representa um registro de propriedade, cobrindo tanto imóveis urbanos (via portais IPTU municipais) quanto rurais (SNCR, CAFIR, IBAMA).

## Pontos de entrada

| Dado disponível | Endpoint | Resultado |
|----------------|----------|-----------|
| CPF | `GET /pessoas/cpf/{cpf}/imoveis` | Imóveis urbanos da pessoa |
| CPF | `GET /pessoas/cpf/{cpf}/rural` | Perfil rural completo |
| CNPJ | `GET /empresas/cnpj/{cnpj}/imoveis` | Imóveis da empresa |
| CNPJ | `GET /empresas/cnpj/{cnpj}/rural` | Rural da empresa |
| CPF/UF | `GET /imoveis/cpf/{cpf}/{uf}` | Filtrado por estado |

<Tip>
  Na rota principal de pessoa (`GET /pessoas/cpf/{cpf}`), os imóveis já vêm incluídos nos campos `imoveis` (urbanos) e `rural`. Use as sub-rotas para consultas mais rápidas.
</Tip>

## Imóveis urbanos

Cobertura de **1.400+ municípios** via integração com portais IPTU.

### Estrutura

| Campo | Descrição | Exemplo |
|-------|-----------|---------|
| `id` | Identificador único | `a1b2c3d4e5f6` |
| `fonte` | Portal de origem | `bh`, `sp-iptu`, `poa` |
| `sql_iptu` | Inscrição municipal | `0123456789` |
| `localizacao.logradouro` | Rua/Avenida | `R HENRIQUE PASSINI` |
| `localizacao.numero` | Número | `748` |
| `localizacao.bairro` | Bairro | `SERRA` |
| `localizacao.cidade` | Cidade | `BELO HORIZONTE` |
| `localizacao.uf` | Estado | `MG` |
| `localizacao.cep` | CEP | `30220-380` |
| `area_terreno` | Área do terreno (m2) | `360.00` |
| `area_construida` | Área construída (m2) | `180.00` |
| `ano_construcao` | Ano de construção | `1995` |
| `tipo_uso` | Uso do imóvel | `RESIDENCIAL`, `COMERCIAL` |
| `valor_venal` | Valor venal (R$) | `450000.00` |
| `valor_iptu` | Valor do IPTU anual (R$) | `2400.00` |

### Proprietário

Cada imóvel inclui dados do proprietário:
- `nome` — nome conforme cadastro municipal
- `documento` — CPF ou CNPJ
- `tipoDocumento` — `CPF` ou `CNPJ`

## Imóveis rurais

O perfil rural (`/cpf/{cpf}/rural`) agrega dados de múltiplas fontes:

| Fonte | Descricao |
|-------|-----------|
| **SNCR** | Sistema Nacional de Cadastro Rural (INCRA) — área, módulos fiscais, detentores |
| **CAFIR** | Cadastro Fiscal de Imóveis Rurais (Receita Federal) — NIRF, situação fiscal |
| **IBAMA** | Fiscalização ambiental — multas, embargos, áreas embargadas |
| **DOF** | Documento de Origem Florestal — transporte de produtos madeireiros |
| **Produtores** | Cadastro de produtores orgânicos |

## Exemplo de uso

```python
import requests

BASE = "https://221b-api.sherlocker.com.br/api/v1"

# Imoveis urbanos via perfil completo
pessoa = requests.get(f"{BASE}/pessoas/cpf/12345678901", params={"token": TOKEN}).json()
imoveis = pessoa.get("imoveis", [])
rural = pessoa.get("rural", {})

# Via sub-rota especifica
resp = requests.get(f"{BASE}/pessoas/cpf/12345678901/imoveis", params={"token": TOKEN}).json()

for im in imoveis:
    loc = im.get("localizacao", {})
    print(f"{loc.get('logradouro')}, {loc.get('numero')} - {loc.get('cidade')}/{loc.get('uf')}")
    print(f"  Valor venal: R$ {im.get('valor_venal', 0):,.2f}")
    print(f"  Area: {im.get('area_terreno', 0)} m2 terreno, {im.get('area_construida', 0)} m2 construido")
```

## Estratégia de levantamento patrimonial

1. **Pessoa alvo** — buscar imóveis urbanos + rurais por CPF
2. **Parentes** — repetir para cônjuge e filhos (imóveis podem estar em nome de familiares)
3. **Empresas** — buscar imóveis dos CNPJs onde a pessoa é sócia
4. **Cruzamento** — comparar endereços de moradia com endereços de propriedade

<Info>
  Consultas de imóveis podem levar alguns segundos por dependerem de portais municipais. Para consultas em lote, use `?enrich=true` para incluir dados de leilão e Caixa Econômica.
</Info>
