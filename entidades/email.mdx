Um **Email** representa um endereço de email associado a uma pessoa física, com classificação automática entre pessoal e corporativo.

## Tipagem

```json
{
  "email": "maria.santos@gmail.com",
  "dominio": "gmail.com",
  "corporativo": false
}
```

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `email` | string | Endereço de email completo |
| `dominio` | string | Domínio extraído do email |
| `corporativo` | boolean | `false` para provedores gratuitos (Gmail, Hotmail, Yahoo, etc.), `true` para domínios próprios |

## Conexões

- **Pessoa** — todo email pertence a um CPF
- **Empresa** — empresas têm email comercial (`correio_eletronico`)
- **Perfis digitais** — emails podem ser usados para buscar perfis em plataformas (GitHub, LinkedIn, etc.)
- **Domínios** — emails corporativos se conectam a domínios registrados

## Endpoints

| Rota | Descrição |
|------|-----------|
| `GET /emails/cpf/{cpf}` | Emails de uma pessoa |
| `GET /emails/email/{email}` | Busca reversa: CPFs por email |
| `GET /perfis/email/{email}` | Perfis digitais vinculados ao email |
