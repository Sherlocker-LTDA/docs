O módulo **Rural** do Sherlocker agrega dados de propriedades rurais, fiscalização ambiental e transporte de produtos florestais a partir de múltiplas fontes federais.

## Pontos de entrada

| Dado disponível | Endpoint | Resultado |
|----------------|----------|-----------|
| CPF | `GET /rural/cpf/{cpf}` | Perfil rural completo |
| CNPJ | `GET /rural/cnpj/{cnpj}` | Perfil rural da empresa |
| Código SNCR | `GET /rural/imovel/{codigo}` | Imóvel rural por código |
| Código IBGE | `GET /rural/municipio/{codIbge}` | Imóveis por município |
| NIRF | `GET /rural/cafir/{nirf}` | Imóvel CAFIR por NIRF |

<Tip>
  Na rota principal de pessoa (`GET /pessoas/cpf/{cpf}`), o campo `rural` já vem incluído com o perfil completo. Use `/rural/cpf/{cpf}` quando precisar apenas dos dados rurais.
</Tip>

## Fontes de dados

### SNCR — Sistema Nacional de Cadastro Rural (INCRA)

Cadastro oficial de imóveis rurais do Brasil.

| Campo | Descrição |
|-------|-----------|
| `codigo_imovel` | Código único do imóvel no SNCR |
| `nome_imovel` | Nome da propriedade |
| `municipio` / `uf` | Localização |
| `area_total` | Área total em hectares |
| `modulos_fiscais` | Quantidade de módulos fiscais |
| `nome_detentor` | Nome do proprietário/detentor |
| `cpf_cnpj` | Documento do detentor |
| `fracao` | Percentual de participação (%) |
| `condicao_detentor` | Proprietário, arrendatário, etc. |

### CAFIR — Cadastro Fiscal de Imóveis Rurais (Receita Federal)

Cadastro tributário vinculado ao ITR (Imposto Territorial Rural).

| Campo | Descrição |
|-------|-----------|
| `nirf` | Número do Imóvel na Receita Federal |
| `area_total` | Área total em hectares |
| `situacao` | Ativo, cancelado, etc. |
| `imune_isento` | Se é imune ou isento de ITR |
| `codigo_sncr` | Vínculo com o SNCR |

### Fiscalização ambiental (IBAMA)

Autos de infração, multas e embargos ambientais.

| Campo | Descrição |
|-------|-----------|
| `numero_auto` | Número do auto de infração |
| `tipo_auto` | Tipo (embargo, multa, apreensão) |
| `descricao_dano` | Descrição do dano ambiental |
| `area_embargada` | Área embargada em hectares |
| `valor_multa` | Valor da multa (R$) |
| `situacao_debito` | Pago, em cobrança, etc. |

### DOF — Transporte Florestal

Guias de transporte de produtos florestais (madeira, carvão, etc.).

| Campo | Descrição |
|-------|-----------|
| `produto` | Tipo de produto florestal |
| `volume` | Volume transportado |
| `valor_total` | Valor total da operação |
| `remetente` / `destinatario` | Partes envolvidas |

### Produtores orgânicos

Cadastro nacional de produtores orgânicos.

| Campo | Descrição |
|-------|-----------|
| `nome_produtor` | Nome do produtor |
| `escopo` | Escopo da certificação |
| `atividade` | Tipo de atividade |
| `situacao` | Status da certificação |

## Perfil rural completo

A rota `GET /rural/cpf/{cpf}` retorna um perfil agregado com:

- **Propriedades SNCR** — todos os imóveis rurais vinculados
- **Propriedades CAFIR** — cadastro fiscal vinculado aos códigos SNCR
- **Transportes** — resumo de operações florestais com volume, valor e parceiros
- **Autorizações** — autorizações de exploração florestal (AUTEX)
- **Infrações** — autos de infração do IBAMA
- **Produtor orgânico** — certificações ativas
- **Resumo** — área total, quantidade de imóveis, valor total de multas, estados

## Exemplo de uso

```python
import requests

BASE = "https://221b-api.sherlocker.com.br/api/v1"

# Perfil rural completo
rural = requests.get(f"{BASE}/rural/cpf/12345678901", params={"token": TOKEN}).json()

resumo = rural.get("rural", {}).get("resumo", {})
print(f"Area total: {resumo.get('area_total_ha', 0)} ha")
print(f"Imoveis SNCR: {resumo.get('qtd_imoveis_sncr', 0)}")
print(f"Infracoes IBAMA: {resumo.get('qtd_infracoes', 0)}")
print(f"Multas: R$ {resumo.get('valor_total_multas', 0):,.2f}")

# Busca por municipio
imoveis = requests.get(f"{BASE}/rural/municipio/3106200", params={"token": TOKEN}).json()
```
