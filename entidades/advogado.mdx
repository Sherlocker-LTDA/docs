Um **Advogado** representa um profissional do direito que atua como representante legal de uma parte em um processo judicial.

## Campos

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `nome` | string | Nome completo do advogado (Title Case) |
| `oab` | string | Número da OAB com UF (ex: `SP123456`, `RJ084902`) |
| `polo` | string | Polo que representa (`ativo`, `passivo`, etc.) |
| `documento` | string \| null | CPF do advogado (pode não estar disponível) |

## Exemplo

```json
{
  "nome": "José Carlos Mendes",
  "oab": "SP123456",
  "polo": "ativo",
  "documento": "987.654.321-00"
}
```
