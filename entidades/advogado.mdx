icon: "scale-balanced"
---

Um **Advogado** representa um profissional do direito que atua como representante legal de uma parte em um processo judicial.

## Campos

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `nome` | string | Nome completo do advogado |
| `documento` | string \| null | CPF do advogado |
| `oab` | string | Número da OAB (ex: SP123456) |

## Exemplo

```json
{
  "nome": "Dr. José Carlos Advogado",
  "documento": "98765432100",
  "oab": "SP123456"
}
```
