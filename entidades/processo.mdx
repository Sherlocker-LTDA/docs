Um **Processo** representa um processo judicial coletado de multiplos tribunais brasileiros. Cada processo contem suas partes (autor, reu), advogados e movimentacoes.

## Campos

| Campo | Tipo | Descricao |
|-------|------|-----------|
| `numero_processo` | string | Numero unificado CNJ (20 digitos) |
| `numero_formatado` | string | Numero formatado (ex: 0001234-56.2020.8.26.0100) |
| `tribunal` | string | Sigla do tribunal (TJ-SP, TRF-2, TRT-2, etc.) |
| `classe_codigo` | number | Codigo da classe processual |
| `classe_nome` | string | Nome da classe (Procedimento Comum Civel, Execucao Fiscal, etc.) |
| `grau` | string | Grau de jurisdicao (G1, G2, REsp, etc.) |
| `data_ajuizamento` | string | Data de ajuizamento |
| `orgao_julgador` | string | Vara ou turma responsavel |
| `assuntos` | string | Assuntos do processo |
| `partes` | Parte[] | Lista de partes envolvidas |
| `advogados` | Advogado[] | Lista de advogados atuantes |
| `movimentacoes` | object[] | Ultimas movimentacoes do processo |

## Segmentacao por classe

| Categoria | Exemplos | Indica |
|-----------|----------|--------|
| Falencia | Recuperacao Judicial, Falencia | Empresa insolvente |
| Execucao | Execucao Fiscal, Titulo Extrajudicial | Cobranca de divida |
| Trabalhista | Reclamacao Trabalhista | Acao de funcionario |
| Criminal | Acao Penal, Execucao Penal | Processo criminal |
| Civel | Procedimento Comum, Indenizacao | Acao generica |

## Endpoints

| Rota | Descricao |
|------|-----------|
| `GET /processos/cpf/{cpf}` | Processos de uma pessoa |
| `GET /processos/cnpj/{cnpj}` | Processos de uma empresa |
| `GET /processos/numero/{numero}` | Busca por numero do processo |
