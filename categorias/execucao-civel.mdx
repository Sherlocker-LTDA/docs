Identifique **execuções cíveis** — cobranças judiciais de dívidas entre particulares.

```bash
curl "https://221b-api.sherlocker.com.br/api/v1/processos/cpf/{cpf}?categorias=execucao-civel&token=SEU_TOKEN"
```

## Quando usar

- **Análise de crédito** — execuções cíveis indicam dívidas não pagas
- **Cobrança** — verificar se o devedor já tem outras execuções em andamento
- **Locação** — verificar histórico de execuções antes de alugar imóvel
