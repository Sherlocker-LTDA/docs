Identifique **execucoes civeis** — cobracas judiciais de dividas entre particulares.

```bash
curl "https://221b-api.sherlocker.com.br/api/v1/processos/cpf/{cpf}?categorias=execucao-civel&token=SEU_TOKEN"
```

## Quando usar

- **Analise de credito** — execucoes civeis indicam dividas nao pagas
- **Cobranca** — verificar se o devedor ja tem outras execucoes em andamento
- **Locacao** — verificar historico de execucoes antes de alugar imovel
