O Sherlocker permite identificar se uma empresa está em **recuperação judicial** ou **falência** usando o parâmetro `categorias` nos endpoints de processos por CPF ou CNPJ.

## Por que monitorar?

- **Análise de crédito** — empresas em recuperação judicial têm risco elevado de inadimplência
- **Due diligence** — antes de fechar contratos, verificar se a contraparte está em processo de falência
- **Monitoramento de carteira** — acompanhar clientes ou fornecedores que entraram em recuperação

## Como filtrar

Adicione o parâmetro `categorias` ao endpoint de processos por CNPJ ou CPF:

### Verificar uma empresa (CNPJ)

```bash
# Somente recuperações judiciais
curl "https://221b-api.sherlocker.com.br/api/v1/processos/cnpj/12345678000199?categorias=recuperacao-judicial&token=SEU_TOKEN"

# Somente falências
curl "https://221b-api.sherlocker.com.br/api/v1/processos/cnpj/12345678000199?categorias=falencia&token=SEU_TOKEN"

# Ambas
curl "https://221b-api.sherlocker.com.br/api/v1/processos/cnpj/12345678000199?categorias=recuperacao-judicial,falencia&token=SEU_TOKEN"
```

### Verificar uma pessoa (CPF)

```bash
curl "https://221b-api.sherlocker.com.br/api/v1/processos/cpf/12345678901?categorias=recuperacao-judicial,falencia&token=SEU_TOKEN"
```

## Categorias disponíveis

| Categoria | Descrição |
|-----------|-----------|
| `recuperacao-judicial` | Recuperações judiciais |
| `falencia` | Falências e insolvência |
| `execucao-fiscal` | Execuções fiscais |
| `execucao-civel` | Execuções cíveis |
| `criminal` | Processos criminais |
| `trabalhista` | Reclamações trabalhistas |
| `familia-sucessoes` | Família e sucessões |

## Exemplo prático: monitoramento de carteira

```python
import requests

TOKEN = "SEU_TOKEN"
BASE = "https://221b-api.sherlocker.com.br/api/v1"

def verificar_carteira(cnpjs: list[str]):
    """Verifica quais empresas da carteira estão em recuperação ou falência."""
    alertas = []

    for cnpj in cnpjs:
        resp = requests.get(
            f"{BASE}/processos/cnpj/{cnpj}",
            params={
                "categorias": "recuperacao-judicial,falencia",
                "token": TOKEN,
            },
        )
        data = resp.json()

        if data.get("total", 0) > 0:
            alertas.append({
                "cnpj": cnpj,
                "processos": data["total"],
                "detalhes": [
                    {
                        "numero": p["numero_formatado"],
                        "classe": p["classe_nome"],
                        "tribunal": p["tribunal"],
                    }
                    for p in data["processos"]
                ],
            })

    return alertas

# Exemplo de uso
carteira = ["12345678000199", "98765432000188", "11222333000144"]
alertas = verificar_carteira(carteira)

for alerta in alertas:
    print(f"ALERTA: {alerta['cnpj']} — {alerta['processos']} processo(s)")
    for d in alerta["detalhes"]:
        print(f"  {d['numero']} ({d['classe']} - {d['tribunal']})")
```
